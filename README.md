# Portus

Master repository for the custom Obsidian **Portus Theme** and **Portus Kanban Plugin**.

## Structure

- `theme/`: Contains the CSS and manifest for the Portus Theme.
- `plugin/`: Contains the main.js, manifest.json, and styles for the Portus Kanban Plugin.

## Installation

### Theme
1. Copy the contents of the `theme` folder to your vault's `.obsidian/themes/Portus Theme/` directory.
2. Enable it in Obsidian Settings > Appearance.

### Plugin
1. Copy the contents of the `plugin` folder to your vault's `.obsidian/plugins/portus-kanban/` directory.
2. Enable it in Obsidian Settings > Community Plugins.
